# Cheats-School
Cheats for school games like Kahoot and Blooket.

For Blooket, just run the extension on the page after you join

For Kahoot, please get the quizid before joining, and run the extension before inputting the join code.
Quizid is usually the last string of gibberish behind the url on your teacher/host screen.

Also, thanks to the different people that I got the script from, go check out their websites and posts.
Their scripts are posted as well, you can just copy everything (Ctrl+a) then open chrome console and paste.
If you don't know where chrome console is, WHY ARE YOU EVEN HERE????
Basically, just right click the page, then click inspect, then switch to console.
if it shows a yellowish error, type in 'allow pasting', then paste again.

For those who do not know how to load a extension, follow these steps;
1. Go to your browser, go to the extensions page.
  - You can do this by either typing in chrome://extensions (for chrome) or clicking the extension button near top right of your browser. (Looks like a jigsaw puzzle piece)
2. Then enable developer mode
3. Find the button that says load unpacked (Unzip the file first if you haven't done already)
4. Then find the file and drop it in
5. You are done!!

btw you can pin it for efficiency

Also you can find the scripts being ran in the files labeled as Blooket.js, kahoot-1.js and kahoot-2.js
